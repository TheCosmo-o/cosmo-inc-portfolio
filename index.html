O mnie

Nazywam się Kasjan i jestem technikiem informatykiem specjalizującym się w sieciach komputerowych, administracji systemami oraz wsparciu technicznym. Mam doświadczenie w konfiguracji sprzętu, tworzeniu stron internetowych i baz danych. Nieustannie rozwijam swoje kompetencje praktyczne, by skutecznie wspierać użytkowników i realizować projekty IT.

Moje umiejętności:

Windows Linux VMware LAN/WAN Cisco TCP/IP DHCP/DNS HTML5 CSS MySQL Python VirtualBox Kopie zapasowe Bezpieczeństwo IT Diagnostyka sprzętu Wsparcie techniczne Microsoft Office Zarządzanie zgłoszeniami Ochrona danych
